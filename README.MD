# TicTacToe using minimax algorithm
A simple implementation of a minimax algorithm to play a game of <a href="https://www.google.com/search?q=TicTacToe&oq=TicTacToe&aqs=chrome..69i57j0l2j69i60l3.2212j0j7&sourceid=chrome&ie=UTF-8">Tic-Tac-Toe</a>
<ul>
<li> AI is unbeatable
<li> Game rules aren't fully inforced. Over-writing existing tile is possible
<li> GUI implementation yet to be made
<li> No alpha-beta pruning implemented yet.
</ul>
